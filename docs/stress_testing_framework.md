This document defines the enterprise stress-testing governance framework implemented within the KRONOS institutional financial-intelligence platform.



The framework supports:

\- systemic escalation monitoring.

Value at Risk (VaR) Framework



Objective



Estimate potential portfolio loss under stressed conditions.





\---



VaR Formula



VaR\_{\\alpha} = \\mu - z\_{\\alpha}\\sigma





\---



VaR Components
