Loss Given Default (LGD) Methodology



Objective



Estimate expected loss severity after borrower default.





\---



LGD Methodology



KRONOS estimates LGD using:



LGD = (Exposure - Recoverable Collateral) / Exposure



Factors considered include:



collateral value,



recovery assumptions,



sector deterioration,



macroeconomic stress conditions.







\---



Exposure at Default (EAD) Methodology



Objective



Estimate total exposure at the time of borrower default.





\---



EAD Methodology



KRONOS estimates EAD using:



EAD = Current Balance + (Undrawn Commitment × CCF)



Where:



CCF = Credit Conversion Factor

VaR Methodology



Value at Risk



KRONOS estimates portfolio loss thresholds using:



VaR\_{\\alpha} = \\mu - z\_{\\alpha}\\sigma



Where:



μ = portfolio return mean



σ = portfolio volatility



z = confidence threshold
