Expected Credit Loss (ECL) Framework



Objective



Estimate expected credit losses under enterprise IFRS9 governance principles.





\---



ECL Formula



ECL = PD \\times LGD \\times EAD





\---



ECL Components



Component	Description



PD	Probability of Default

LGD	Loss Given Default

EAD	Exposure at Default







\---



Probability of Default (PD)



Objective



Estimate the probability of borrower default within the defined risk horizon.





\---



PD Inputs



KRONOS incorporates:



borrower deterioration,



delinquency escalation,



utilization volatility,



macroeconomic instability,



behavioral deterioration.







\---



Loss Given Default (LGD)



Objective



Estimate loss severity after borrower default.





\---



LGD Formula



LGD = \\frac{Exposure - Recovery}{Exposure}





\---



LGD Components



KRONOS evaluates:



collateral recovery,



exposure seniority,



sector deterioration,



macroeconomic stress.







\---



Exposure at Default (EAD)



Objective



Estimate total exposure at default realization.





\---



EAD Formula



EAD = Current\\ Balance + (Undrawn\\ Commitment \\times CCF)





\---



EAD Components



KRONOS incorporates:



current balances,



revolving utilization,



undrawn commitments,



conversion assumptions.
